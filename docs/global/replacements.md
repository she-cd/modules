# 不合适应用替换清单

以下条目因设备普适性、权限敏感或实现成本与收益不匹配被替换：

- 66 HTTP 抓包（轻） → HTTP 请求调试台：避免系统级抓包权限，主打构造与调试请求。
- 83 语音同传（简） → 短句语音翻译：改为分段识别+翻译，避免持续流式高成本与时延。
- 95 居家运动计划 → 动作计时与训练计数：剔除视频内容库，保留计时与组间引导。
- 117 酒店拍照量尺 → 照片比例尺标注器：无需 AR 引擎，手动标尺+参考物比例。
- 118 公共交通等车计时 → 通勤出发提醒：不依赖实时公交数据，基于用户时间表与缓冲策略。
- 178 红外遥控库（离线码） → 蓝牙遥控宏定义：部分机型无红外，改为通用蓝牙宏。
- 49 手持稳像（简版） → 视频画幅裁切助手：放弃重稳像算法，提供构图裁切与简单平滑。

以上替换已体现在 `catalog/apps_catalog.json` 中的名称、功能与依赖。
